Currency Converter

A simple and modern currency converter web application built with HTML, CSS, and JavaScript, featuring real-time exchange rates via a public API.

This project was created as a final coursework project and fully meets all mandatory requirements.

Features

Real-time currency conversion using an external API

Automatic conversion when changing amount or currencies

Swap currencies with one click

Adjustable amount input

Accurate exchange rate calculation

Minimalistic modern UI (based on provided Figma design)

Responsive layout (works on desktop and mobile)

Technologies Used

HTML5 — application structure

CSS3 — styling and layout

JavaScript (ES6) — application logic

Exchange Rate API — live currency rates

API Integration

The project uses a public currency exchange API to fetch up-to-date exchange rates.

Example API request:

https://api.exchangerate.host/latest?base=USD

The API is used to:

retrieve current exchange rates

calculate converted values dynamically
